# Objetivo del proyecto

Crear una aplicación movil de contenido audio visual llamada Snack, usando React Native, Expo, React Hook Form, Zod y Nativewind.

El proyecto ya está creado y con nativewind, React Hook Form, Zod y TanStack Query instalados, pero sin estilos ni funcionalidades. El objetivo es implementar el diseño y las funcionalidades siguiendo exactamente los diseños presentados en las referencias que serán proporcionadas que se encuentran en la carpeta "Antigravity" dentro del proyecto.
