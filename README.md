# Welcome to your Expo app 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

3. Test in mobile dispositve
- Download the Expo Go app on your phone (search it in Google Play or App Store)
- When the app is running, scan the QR code with the Expo Go app on your phone or enter the url in your mobile
